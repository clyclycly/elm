## 使用步骤

``` bash
# 下载
git clone https://github.com/clyclycly/elm

# 安装依赖
npm install

# 运行开发环境
npm run dev

# 构建生产环境
npm run build

```

## 技术栈
 + vue1.0：基础框架
 + vue-router1.0：路由跳转
 + es6：采用部分es6特性，大大简化了写法
 + webpack：vue-cli基于webpack，修改了部分配置
 + style：写起css来十分方便，需要安装style-loader，另外使用自定义其中的自定义函数配合rem来适配移动端设备简直不要太方便
